Prerequisites
=============

Before proceeding with this module, ensure you have the following:

- Python 3.8 or higher installed
- A code editor (such as VS Code, PyCharm, or similar)
- Git installed and configured
- Access to the command line (Terminal, Command Prompt, or PowerShell)
- Basic familiarity with Python programming

If you need help installing any of these tools, refer to the official documentation or your course setup guide.
